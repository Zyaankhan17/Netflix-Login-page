 # 🎬 Netflix Login Page Clone

A simple **Netflix Login Page Clone** built using **HTML5** and **CSS3**. This project recreates the login page of Netflix with a clean and responsive design to practice frontend development and CSS styling.

## 🚀 Features

* Netflix-inspired login page
* Responsive layout
* Clean and modern UI
* Background image with overlay
* Styled input fields and buttons
* "Remember Me" checkbox
* "Need Help?" link
* Footer section with useful links

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
Netflix-Login-Clone/
│── index.html
│── style.css
│── images/
│── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/netflix-login-clone.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
